# ashu

Official digital portfolio of **Dr. Ashutosh Gupta** — Professor of Chemistry, Udai Pratap College, Varanasi (Computational & Organic Chemistry).

A futuristic, fully mobile-responsive 6-page static website (HTML / CSS / JS — no build step).

## Pages
- `index.html` — Home / Hero
- `about.html` — Academic Profile (education, fellowships)
- `research.html` — 29 Publications with decade filter
- `awards.html` — Conferences attended & organized
- `teaching.html` — Student's Corner / YouTube lectures
- `contact.html` — Contact, phone, WhatsApp & form

## Run locally
Open `index.html` in any browser — it works fully offline.

## Deploy
Drag the folder onto [Netlify Drop](https://app.netlify.com/drop) or enable GitHub Pages.
